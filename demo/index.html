MS-DOC Viewer

MoonBit parser compiled to multiple targets. Each demo uses a different build output.

JavaScript Browser + Node
Direct ESM import of MoonBit-compiled JS. Runs on the main thread. Works in any modern browser and Node.js.
WASM Near-Native Speed
MoonBit compiled to WebAssembly linear memory. Binary data passed through WASM memory for maximum throughput.
Web Worker Non-Blocking
Same MoonBit JS build running in a Web Worker thread. Parsing happens off the main thread — UI stays responsive.
WASM-GC Browser + Runtime
MoonBit compiled to WASM with Garbage Collection. Uses native JS string builtins (Chrome 112+) for zero-copy interop.
JS Reference Original JS
Original JavaScript implementation — for side-by-side comparison with MoonBit targets.
