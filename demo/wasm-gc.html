How It Works

MoonBit compiles to WASM-GC with use-js-builtin-string: true, making MoonBit's String type equivalent to JavaScript's native string. The module imports wasm:js-string builtins (Chrome 112+, Firefox 120+, Safari 16.4+) for zero-copy string interop.

Binary .doc data is hex-encoded on the JS side, passed as a native string, and the WASM-GC function returns CSS+HTML as a native JS string.

JS Instantiation

const { instance } = await WebAssembly.instantiate(bytes, {}, {
  builtins: ['js-string'],
  importedStringConstants: '_'
});
const result = instance.exports.render_ms_doc_wasm(hexString);
// result is a native JS string: "CSS|||HTML"
